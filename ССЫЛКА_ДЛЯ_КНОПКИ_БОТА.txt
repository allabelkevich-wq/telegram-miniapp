  РАБОЧАЯ ССЫЛКА ДЛЯ TELEGRAM MINI APP (кнопка меню бота)
═══════════════════════════════════════════════════════════════

Актуальный URL — в Render Dashboard или:
  https://<твой-сервис>.onrender.com/api/miniapp-url

Варианты по имени сервиса:
  • yupsoul-bot → https://yupsoul-bot.onrender.com
  • telegram-miniapp-ar09 → https://telegram-miniapp-ar09.onrender.com

(без слеша в конце, без пробелов)

───────────────────────────────────────────────────────────────
Приложение раздаётся с Render. Бот при каждом /start и при деплое
автоматически обновляет кнопку меню на этот URL.
───────────────────────────────────────────────────────────────

ВАРИАНТ 1 — Автоматически (рекомендуется):
  Напиши боту /start — кнопка обновится сама.

ВАРИАНТ 2 — Вручную через @BotFather:
  1. Открой @BotFather в Telegram
  2. Напиши команду: /setmenubutton
  3. Выбери своего бота
  4. Configure menu button
  5. Вставь URL (см. варианты выше — твой .onrender.com)
  6. Текст кнопки: YupSoul (или «Открыть приложение»)

ВАРИАНТ 3 — Через Bot Settings (если /setmenubutton не сработал):
  1. @BotFather → /mybots → твой бот
  2. Bot Settings → Menu Button → Configure menu button
  3. Вставь свой URL (как в вариантах выше)

───────────────────────────────────────────────────────────────
Проверка актуального URL:
  Открой в браузере: https://твой-сервис.onrender.com/api/miniapp-url
  В ответе будет url — это и есть рабочая ссылка для кнопки.
