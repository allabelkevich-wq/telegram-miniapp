Ссылка для кнопки меню в @BotFather (если слетела):

https://telegram-miniapp-ar09.onrender.com/

Приложение раздаётся с Render (бот уже отдаёт Mini App с сервера).
GitHub Pages не используется.

Важно: на бесплатном плане Render сервис «засыпает» после ~15 мин без активности.
При первом открытии может занять 30–60 сек — это нормально.

Как вставить:
1. Telegram → @BotFather → /mybots → твой бот
2. Bot Settings → Menu Button → Configure menu button
3. Вставить ссылку выше (без пробелов)
4. Текст кнопки: Открыть приложение (или YupSoul)
