В Render добавляй переменные по одной. Для каждой строки ниже:
1) Нажми "Add Environment Variable"
2) В поле Key вставь ровно то, что написано в кавычках (одну строку)
3) В поле Value скопируй значение из своего файла bot/.env

Key №1 (скопируй в поле Key):
BOT_TOKEN

Key №2:
SUPABASE_URL

Key №3:
SUPABASE_SERVICE_KEY

Key №4:
ADMIN_TELEGRAM_IDS
(Value: твой Telegram ID, например 619065619 — без кавычек)

Key №5:
MINI_APP_URL

Key №6:
DEEPSEEK_API_KEY

Key №7:
SUNO_API_KEY

Key №8:
HOT_WEBHOOK_SECRET

Key №9:
HOT_ITEM_ID_DEFAULT

Key №10:
RENDER_EXTERNAL_URL
(Value: https://telegram-miniapp-ar09.onrender.com — без слэша в конце)

---
Итого: 10 переменных. Значения (Value) бери из bot/.env — там такие же названия слева от знака =

ВАЖНО для HOT оплат:
- HOT_WEBHOOK_SECRET: получить в личном кабинете HOT Pay
- HOT_ITEM_ID_DEFAULT: ID платёжной ссылки из HOT Pay (создать в админке)
- RENDER_EXTERNAL_URL: обязательно для корректной работы мини-приложения
