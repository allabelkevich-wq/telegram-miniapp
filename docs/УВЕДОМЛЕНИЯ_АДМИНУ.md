# Уведомления админу о новых заявках

Когда пользователь отправляет заявку из Mini App (кнопка «Отправить заявку во Вселенную»), бот шлёт **тебе в личный чат с ботом** сообщение о новой заявке.

## Что нужно, чтобы уведомления приходили

1. **Ты должен один раз написать боту в Telegram**  
   Открой @Yup_Soul_bot и отправь **/start** (или любое сообщение). Иначе Telegram не даёт боту писать тебе первым.

2. **В `bot/.env` указан твой ID**  
   Строка: `ADMIN_TELEGRAM_IDS=619065619` (подставь свой ID). Узнать ID: напиши в Telegram боту @userinfobot.

3. **Бот запущен из папки проекта**  
   В терминале: `cd /Users/arhan/telegram-miniapp/bot && npm start`. После изменений в коде бота нужен перезапуск (Ctrl+C, затем снова `npm start`).

## Что видно в консоли

При новой заявке в консоли бота должны появиться строки:
- `[Заявка] ...`
- `[Уведомление] Отправляю админам: 619065619`
- `[Уведомление] Доставлено админу 619065619`

Если видишь `[Уведомление админу] 619065619 ...` с ошибкой — бот не смог отправить сообщение (часто из‑за того, что ты ещё ни разу не писал боту).

## Если ничего не изменилось

- **Уведомления не приходят** — выполни пункты 1–3 выше и отправь тестовую заявку из приложения. Проверь консоль бота на строки `[Уведомление]`.
- **Изменения в приложении (кнопки, страница Забота)** — они подхватываются только после **пуша в GitHub** (приложение открывается с GitHub Pages). Локальные правки в `index.html` в Telegram не видны, пока не задеплоишь.
- **Админка /admin зависает** — перезапусти бота (в коде есть таймаут 12 сек). Если после перезапуска всё так же — напиши, разберём дальше.
